L'application Rappels fonctionne de la manière suivante :

- Le MainActivity affiche une liste de rappels, se trouvant sur une FireBase.

- Le bouton à droite permet de créer des rappels, en entrant le titre, le contenu et la date.

- Le bouton à gauche permet de modifier un rappel
=> Le bouton valider permet de auto remplir les champs pour modifier facilement le rappel
=> Le bouton supprimer sert à supprimer à partir du titre le rappel
